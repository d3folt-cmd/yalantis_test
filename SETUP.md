### How do I get set up a script? ###

* Clone repository
* Goto download folder
* Install virtual env:
    ```bash
    pip install virtualenv
    virtualenv venv
    source venv/bin/activate
    ```
* Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
* Run app.py script:  
    ```bash
    python3 app.py
    ```
